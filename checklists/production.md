# Production Checklist

Complete these steps during recording.

## Before Recording

### Camera Setup (if using)
- [ ] Camera powered and recording to correct storage
- [ ] Frame composed correctly
- [ ] Focus set
- [ ] Exposure/white balance correct
- [ ] External mic connected and levels set

### Screen Capture Setup (if using)
- [ ] Recording software ready (OBS/ScreenFlow/etc.)
- [ ] Correct screen/window selected
- [ ] Resolution set (1080p or 4K)
- [ ] Frame rate set (30 or 60fps)
- [ ] Audio input selected
- [ ] Recording destination has space

### Environment
- [ ] Do Not Disturb enabled on all devices
- [ ] Unnecessary apps closed
- [ ] Desktop clean
- [ ] Notifications disabled
- [ ] Phone silenced/away

## During Recording

### Camera Recording
- [ ] Clap or sync marker at start (for multi-source sync)
- [ ] Energy up from the start
- [ ] Speak clearly and at good pace
- [ ] Eye contact with lens
- [ ] Multiple takes OK - don't stress mistakes

### Screen Recording
- [ ] Mouse movements deliberate and visible
- [ ] Pause briefly when switching contexts
- [ ] Narrate what you're doing
- [ ] Zoom in on important details
- [ ] Keep terminal/code visible and readable

### General Tips
- [ ] Take breaks between sections if needed
- [ ] Note any mistakes to edit later (timecode if possible)
- [ ] Get extra B-roll if opportunity arises
- [ ] Record room tone (10 seconds of silence)

## After Recording

### Immediate
- [ ] Verify all files recorded successfully
- [ ] Playback check - audio and video quality OK
- [ ] Backup raw footage immediately
- [ ] Note any issues or reshoot needs

### File Organization
- [ ] Move raw files to episode folder
- [ ] Name files descriptively
- [ ] Create folder structure:
  ```
  episode-folder/
  ├── raw/
  │   ├── camera/
  │   └── screen/
  ├── audio/
  ├── exports/
  └── project-files/
  ```

### Initial Review
- [ ] Quick watch-through
- [ ] Note major edit points
- [ ] Identify any pickup shots needed
- [ ] Confirm nothing needs to be re-recorded
