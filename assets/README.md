# Shared Assets

This directory contains reusable media assets for content production.

## Folder Structure

- **branding/** - Logos, brand guides, fonts, and brand-related assets
- **intros/** - Intro video clips and animations
- **outros/** - Outro/end screen clips
- **overlays/** - Lower thirds, graphics overlays, and visual elements
- **music/** - Background music tracks and themes
- **sound-effects/** - Transition sounds, stingers, and audio effects
- **thumbnails/** - Thumbnail templates, backgrounds, and design elements

## Usage

Reference these assets in your episode metadata for consistent branding and production quality.
