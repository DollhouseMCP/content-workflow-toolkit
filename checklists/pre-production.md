# Pre-Production Checklist

Complete these steps before recording.

## Planning
- [ ] Topic identified and validated (is there audience interest?)
- [ ] Research completed
- [ ] Target audience defined
- [ ] Video length estimated
- [ ] Recording style decided (camera/screen/mixed)

## Script Development
- [ ] Outline created
- [ ] Script written using `templates/script-template.md`
- [ ] Hook is compelling (first 30 seconds)
- [ ] Content flows logically
- [ ] Call-to-action included
- [ ] Script reviewed/edited

## Technical Prep
- [ ] Code samples prepared and tested
- [ ] Demo environment set up
- [ ] All dependencies installed/working
- [ ] Terminal/IDE configured for recording
  - [ ] Font size increased
  - [ ] Theme is readable on video
  - [ ] Notifications disabled

## Visual Assets
- [ ] Diagrams/graphics created (if needed)
- [ ] B-roll list prepared
- [ ] Thumbnail concept ready
- [ ] Intro/outro assets available

## Metadata Prep
- [ ] `metadata.yml` created from template
- [ ] Title drafted (hook + keywords)
- [ ] Description drafted
- [ ] Tags researched
- [ ] Thumbnail text planned

## Environment Check
- [ ] Recording space clean/organized
- [ ] Lighting tested
- [ ] Audio tested (no echo/background noise)
- [ ] Camera position set
- [ ] Teleprompter ready (if using)

## Final Review
- [ ] Read script aloud once
- [ ] Time estimate still accurate
- [ ] All assets gathered in episode folder
- [ ] Ready to record
