# Brand Guide

## Color Palette

- Primary: #2563eb (Blue)
- Secondary: #64748b (Slate)
- Background: #0f172a (Dark Blue)
- Success: #10b981 (Green)
- Warning: #f59e0b (Orange)
- Error: #ef4444 (Red)

## Typography

- Font Family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- Headings: 600 weight
- Body: 400 weight

## Usage

Apply these colors consistently across all content and materials.
