# Post-Production Checklist

Complete these steps after recording.

## Editing

### Assembly
- [ ] Import all footage to editing software
- [ ] Sync multi-source footage (if applicable)
- [ ] Create rough cut (remove obvious mistakes)
- [ ] Arrange in script order
- [ ] Review pacing

### Fine Cut
- [ ] Trim dead air and long pauses
- [ ] Cut filler words (um, uh, like)
- [ ] Add J-cuts and L-cuts for flow
- [ ] Verify no jump cuts are jarring
- [ ] Check total runtime against target

### Audio
- [ ] Normalize audio levels (-14 LUFS for YouTube)
- [ ] Remove background noise
- [ ] Add room tone to fill gaps
- [ ] Smooth audio transitions
- [ ] Check for pops/clicks/distortion

### Visuals
- [ ] Color correct/grade footage
- [ ] Match multiple camera sources
- [ ] Add intro sequence
- [ ] Add outro sequence
- [ ] Insert B-roll where planned

### Graphics & Text
- [ ] Add lower thirds (if needed)
- [ ] Insert code/terminal overlays
- [ ] Add diagrams/illustrations
- [ ] Ensure text is readable on mobile
- [ ] Add subscribe/like animations (if using)

### Captions
- [ ] Generate transcript (auto or manual)
- [ ] Review and correct transcript
- [ ] Time-align captions
- [ ] Export .srt file

## Export

### Video Export
- [ ] Export at target resolution (1080p/4K)
- [ ] Use YouTube-recommended codec (H.264)
- [ ] Check file size is reasonable
- [ ] Verify no audio/video sync issues

### Assets Export
- [ ] Export thumbnail (1280x720 min)
- [ ] Export caption file (.srt)
- [ ] Export audio-only version (if needed for podcast)

## Quality Check

### Final Review
- [ ] Watch entire video at 1x speed
- [ ] Check audio is clear throughout
- [ ] Verify no encoding artifacts
- [ ] Confirm captions sync correctly
- [ ] Get second opinion if possible

### Accessibility
- [ ] Captions are accurate
- [ ] Visual information is also spoken
- [ ] Color contrast is sufficient
- [ ] Audio descriptions (if needed)

## Thumbnail

- [ ] Design follows brand guidelines
- [ ] Text is readable at small size
- [ ] Image is compelling/clickable
- [ ] Saved at correct dimensions (1280x720)
- [ ] File size under 2MB

## Final Files

- [ ] Video file named correctly
- [ ] Thumbnail ready
- [ ] Caption file ready
- [ ] Description written
- [ ] Tags finalized
- [ ] All files in `exports/` folder
